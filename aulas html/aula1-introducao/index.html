O CSS

Folha de estilo em cascata, serve para formartar as TAGS definidas no HTML.

As formatação incluem cor, alinhamento, altura, largura, aparencia de objetos em geral.

Principais TAGS

  • A tag <p> serve para definir um parágrafo.
  • A tag <h1>serva para um titulo principal.
  • A tag <br>serve para quebrar linha
  • A teg <code>serve para mostrar um codigo
  • A teg <ul>seve para criar ele e uma lista não ordernada
